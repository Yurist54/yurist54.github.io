## Публикация на GitHub Pages
1. Создайте репозиторий и загрузите сюда все файлы (ветка `main`).
2. GitHub → Settings → Pages
3. Source: Deploy from a branch
4. Branch: `main` / folder: `/root`
5. Сохраните — GitHub выдаст ссылку вида: `https://<username>.github.io/<repo>/`
